# Cursor Memory - Thông tin quan trọng cần nhớ

## Terminal Commands
- **PowerShell**: Không dùng dấu `&&`, phải dùng dấu `;` để nối lệnh
  - ❌ Sai: `cd flutter-car-scanner && git status`
  - ✅ Đúng: `cd flutter-car-scanner; git status`

## Git Workflow
- **Nhánh công ty**: Khi người dùng nói "push nhánh công ty" nghĩa là push cả 2 nhánh:
  1. `add-feature-demo` → `company/add-feature-demo`
  2. `main` → `company/main`
  
- **Remote repositories**:
  - `origin`: https://github.com/Phatjiro/CSA_PRO.git
  - `company`: https://github.com/Akiranaka/CSA-AppDev.git
